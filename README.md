## 🌿 후기
이 애플리케이션을 구현하기 전까지 아키텍처란 개념에 대해서 크게 주시하지 않았습니다.<br />
이번 공부를 통해 애플리케이션 아키텍처에 대한 중요성을 알게 되었다. <br />
아직은 모놀리식 서비스 아키텍처의 구조보다 익숙지 않아 낯설게 느껴집니다.
아직 인지하지 못한 부분들이 많아 추후 보완해야 할 내용을 공부할 예정이고, 아키텍처에 대해 지속해서 공부할 생각입니다.
